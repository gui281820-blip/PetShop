## Sobre o sistema

O PetShop Zona Sul é uma plataforma web desenvolvida para apresentar produtos, serviços e permitir o agendamento de atendimento para pets.

---

## Página Inicial

Ao acessar o sistema, o usuário encontra a página inicial do PetShop.

Nesta página é possível visualizar:

* Apresentação da empresa.
* Carrossel de imagens.
* Menu de navegação para acesso às demais páginas.

---

## Página de Produtos

Na página Produtos são exibidos os itens comercializados pelo PetShop.

Os produtos estão organizados nas seguintes categorias:

* Acessórios
* Rações
* Higiene e Limpeza

Cada produto apresenta:

* Imagem
* Nome
* Descrição
* Valor

---

## Página de Serviços

Na página Serviços o usuário pode consultar os serviços oferecidos pelo PetShop.

Serviços disponíveis:

* Banho e Tosa
* Banho e Tosa com Tele-Busca

Também são apresentados os valores correspondentes a cada serviço.

---

## Página de Agendamento

A página de Agendamento permite cadastrar o cliente, o pet e solicitar um atendimento.

### Dados do Cliente

Devem ser informados:

* Nome
* Endereço
* CPF
* Sexo
* Telefone
* E-mail

### Dados do Pet

Devem ser informados:

* Nome do Pet
* Raça
* Idade

### Agendamento

O usuário deve selecionar:

* Serviço desejado
* Método de atendimento (tele-busca ou entrega no local)
* Data
* Horário

Após preencher as informações, basta clicar em "Confirmar Agendamento".

O sistema exibirá uma mensagem de confirmação do agendamento.

---

## Página de Contato

Nesta página estão disponíveis as informações de contato do PetShop:

* Telefone
* E-mail
* Endereço

---

## Navegação

O menu superior permite acessar rapidamente qualquer página do sistema:

* Início
* Produtos
* Serviços
* Agendamento
* Contato

Todas as páginas permanecem interligadas por meio deste menu.
